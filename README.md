# Vrinda Store Annual Report 2022 — Sales Dashboard

## Project Overview
An end-to-end sales analysis of Vrinda Store's 2022 performance,
built using Microsoft Excel. Covers revenue trends, customer
demographics, geographic performance, and channel-wise sales.

## Key Insights
- March was the peak month with ₹1.93M in revenue
- 64% of buyers are women; adult women are the highest-value segment
- Maharashtra and Karnataka are the top-performing states
- Amazon holds 35% of channel share — a concentration risk
- 92% delivery rate with 8% returns/cancellations

## Tools Used
- Microsoft Excel (Pivot Tables, Charts, Slicers, Dashboard)
